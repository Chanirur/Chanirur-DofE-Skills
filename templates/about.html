Introduction

Welcome to my personal finance website, developed as part of the Duke of Edinburgh Award. My goal id to provide an easy to use and accessible personal finance tool to people of all backgrounds.

Why I chose this project

I chose this project to contribute something meaningful for others. Combining my interest for programming and disire to help others manage their finances, I chose to build this project.

Technologies Used

This web app is built using industry standard technologies.

  • Backend: Flask (Python Framework) for handling server-side logic.
  • Database: SQLAlchemy to query a SQLite3 database hosted in the cloud.
  • Frontend: HTML, CSS and JavaScript for a clean and user-friendly interface.
