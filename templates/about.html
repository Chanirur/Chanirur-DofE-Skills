About

Introduction

Welcome to my personal finance website, developed as part of the Duke of Edinburgh Award. My goal is to provide an easy to use and accessible personal finance tool to people of all backgrounds.

Technologies Used

This web app is built using industry standard technologies.

  • Backend: Flask (Python Framework) for handling server-side logic.
  • Database: SQLAlchemy to query a SQLite3 database hosted in the cloud.
  • Frontend: HTML, CSS and JavaScript for a clean and user-friendly interface.
